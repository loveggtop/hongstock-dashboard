# 보유종목 통합분석 — 사용법

이 폴더의 증권사 체결 알림(카카오톡 CSV)과 NH 차트분석 AI 캡처 이미지를 합쳐
`통합분석.html` 대시보드를 만들어 주는 **`/mystock` 스킬** 사용 안내입니다.
스킬은 Claude Fable 5 모델로 실행됩니다.

> 스킬 이름 변경: 처음 만들 때의 `/포트폴리오` 명령어는 2026-07-06에 **`/mystock`** 으로
> 변경되었습니다. 이전 이름은 더 이상 동작하지 않습니다.

---

## 1. 기본 실행 (업데이트)

Claude Code를 이 폴더에서 열고 아래 명령어만 입력하면 됩니다.

```
/mystock
```

실행하면 자동으로:

1. 폴더의 `KakaoTalk_Chat_*.csv` 전체를 다시 파싱해 보유종목·평균단가·실현손익 재계산
2. **아직 판독하지 않은 새 이미지만** 골라 지지선·저항선·매매전략 판독
3. `통합분석.html` 재생성 후 브라우저로 열기
4. GitHub 저장소가 연결돼 있으면 `docs/index.html` 갱신 → 커밋 → 푸시 (GitHub Pages 자동 반영)

특정 요청이 있으면 뒤에 붙입니다. 예: `/mystock 천보 위주로 자세히`

## 2. 새 정보 넣는 방법

| 새 정보 | 하는 일 |
|---|---|
| 새 체결 내역 | 카카오톡에서 증권사 채팅방 → 대화 내보내기(CSV) → 이 폴더에 저장 |
| 새 AI 분석 | NH 앱 차트분석 AI 화면 캡처 → 이 폴더에 저장 (카카오톡 이미지 저장 그대로 OK) |

파일을 넣은 뒤 `/mystock` 를 실행하면 반영됩니다.
파일명 규칙은 따로 없지만 CSV는 `KakaoTalk_Chat_...csv` 형식이어야 자동 인식됩니다.

## 3. 결과물

| 파일 | 내용 |
|---|---|
| `통합분석.html` | 통합분석 대시보드 (더블클릭으로 브라우저에서 열림) |
| `docs/index.html` | GitHub Pages 배포용 사본 |
| `.claude/data/trades.json` | 파싱된 체결 내역·보유종목 데이터 |
| `.claude/data/image_analysis.json` | 이미지 판독 결과 (판독 완료 목록) |

대시보드 구성: KPI(보유종목·매입금액·평가손익·실현손익) → 종목별 손익률 차트 →
종목 카드(지지/저항 대비 현재가·평단 위치, NH AI 매수/매도/손절 전략) → 전체 보유내역 표.

## 4. GitHub Pages 확인

배포가 완료되면 휴대폰·PC 어디서든 아래 주소로 볼 수 있습니다.

```
https://loveggtop.github.io/hongstock-dashboard/
```

⚠️ **공개 URL입니다.** 보유종목·수량·손익이 주소를 아는 누구에게나 보입니다.
원본 CSV·캡처 이미지(계좌번호 일부, 실명 포함)는 `.gitignore`로 차단되어 **올라가지 않습니다**.
공개를 중단하려면: GitHub 저장소 → Settings → Pages → 비활성화, 또는 저장소 삭제.

## 5. 수동 실행 (Claude 없이)

스크립트만 직접 돌릴 수도 있습니다 (새 이미지 판독은 안 됨):

```bash
cd ~/agent/Claude/종목매수예측/종목매수가현황
python3 .claude/skills/mystock/scripts/parse_trades.py .      # 1. 체결 재파싱
python3 .claude/skills/mystock/scripts/build_dashboard.py .   # 2. 대시보드 재생성
open 통합분석.html
```

## 6. 문제 해결

- **새 CSV가 반영 안 됨** → 파일명이 `KakaoTalk_Chat_` 으로 시작하는지 확인
- **새 이미지가 반영 안 됨** → `/mystock` 실행 시 Claude가 판독 (수동 스크립트로는 불가)
- **다른 증권사 알림 형식** → `/mystock` 실행 후 "○○증권 형식도 파싱해줘" 라고 요청
- **GitHub 푸시 실패** → `gh auth status` 로 로그인 상태 확인, 필요 시 `gh auth login`

## 7. 유의사항

- 현재가는 **캡처 시점 기준**입니다. 실시간 시세가 아닙니다.
- 매매전략 문구는 NH 차트분석 AI 캡처 내용을 옮긴 것으로, 투자 판단의 참고용입니다.
- 평균단가·실현손익은 평균단가법 기준이며, 수수료·세금은 반영되지 않았습니다.
